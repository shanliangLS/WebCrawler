# WebCrawler
WebCrawler  
网络爬虫网站
功能导航：  
![Image text](https://github.com/shanliangLS/WebCrawler/blob/master/screenShot/%E5%AF%BC%E8%88%AA%E6%A0%8F.gif)  
  
用户设置：修改密码，上传头像，上传背景图片  
  
爬虫策略：添加和修改爬取策略、入口网站  
  
主题管理：查看、创建和修改主题  
  
任务管理：查看、创建、开始和删除任务  
  
爬取数据：查看爬取后的数据并可查看详细信息，导出数据、生成摘要、热词提取
